<title>网页响应式设计</title>

grid布局是支持网页响应式设计的, 但是注意单位只能使用 fr/rem或者百分比

网页响应式设计主要是用到媒体查询功能, 语法如下:

    @media (min-width: 600px) and (max-width: 800px) {
        body {
            ...
        }
        .class {
            ...
        }
    }

当然媒体查询同样可以查询到高度和横竖屏状态

    @media (min-height: 800px) and (orientation: portrait[竖屏]) {
        ...
    }
    @media (min-height: 800px) and (orientation: landscape[横屏]) {
        ...
    }
