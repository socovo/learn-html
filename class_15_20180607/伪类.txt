<title>伪类</title>

CSS伪类是用来添加选择器的一些特殊效果

语法:

    选择器:伪类名 {
        属性: 属性值;
    }
    CSS类也可以使用伪类
    选择器.class:伪类名 {
        属性: 属性值;
    }

anchor伪类:

    链接的不同状态
    a:link      未访问的链接
    a:visited   已访问的链接
    a:hover     鼠标滑过的链接
    a:active    鼠标点击的链接

    使用时严格按照顺序书写

前讲回顾:

    ID选择器        #
    类选择器        .
    通配符选择器    *

群组选择器: 匹配所有逗号隔开的指定标签或带有指定类名的指定标签

    标签1, 标签2, 标签3.类名 {
        属性: 属性值;
    }

后代选择器: 选择存在后代关系的所有指定子级元素

    爷级/父级 子级 {
        属性: 属性值;
    }

子元素选择器: 选择所有严格父子级关系的所有指定子级元素

    父级 > 子级 {
        属性: 属性值;
    }

紧邻兄弟选择器: 选择紧跟子级的第一个且仅有一个指定兄弟元素

    父级 子级 + 子级兄弟元素 {
        属性: 属性值;
    }

    注意: 可连续使用, 但只作用于最后一个元素, 如 p + p + p + p 只作用于最后一个p元素

属性选择器: 选择所有带有指定属性的元素

    [属性] {
        属性: 属性值;
    }
    [属性 = 属性值] {
        属性: 属性值;
    }
    [属性 ^= 属性值开始字符] { /* 选择所有属性值为指定开始字符的元素 */
        属性: 属性值;
    }
    [属性 ~= 属性值结尾字符] { /* 选择所有属性值为指定结尾字符的元素 */
        属性: 属性值;
    }

注意:

    伪类要放在伪元素之前, 例:
    :hover 要放在 ::after 之前
